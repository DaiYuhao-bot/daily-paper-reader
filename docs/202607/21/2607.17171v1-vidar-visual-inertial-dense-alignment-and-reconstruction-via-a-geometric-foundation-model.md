## 摘要
单目基础模型能提供密集几何，但通常缺乏稳定的度量尺度。本文提出VIDAR，一种将SVO+IMU里程计与Depth Anything 3结合的视觉-惯性密集重建框架。VIDAR将视觉-惯性前端作为度量锚点：它提供相机位姿、尺度以及用于跨时间对齐密集基础模型预测的一致世界坐标系。然后，基础模型贡献详细的局部几何信息，这些信息被融合到全局重建中。我们研究了位姿条件DA3和一种解耦对齐策略。在EuRoC上，位姿注入将尺度误差降低至约1%，并达到0.463的平均F@0.10；解耦混合方法无需真实位姿即可将其提升至0.676。在EuRoC和TUM RGB-D上的结果表明，VIDAR是一条实用的度量密度单目重建路线。

## Abstract
Monocular foundation models provide dense geometry but usually lack a stable metric scale. This paper presents VIDAR, a visual-inertial dense reconstruction framework that couples SVO+IMU odometry with Depth Anything 3. VIDAR uses the visual-inertial front end as a metric anchor: it provides camera poses, scale, and a consistent world frame for aligning dense foundation-model predictions across time. The foundation model then contributes detailed local geometry that is fused into a global reconstruction. We study both pose-conditioned DA3 and a decoupled alignment strategy. On EuRoC, pose injection reduces scale error to about 1\% and reaches 0.463 mean F@0.10; the decoupled hybrid improves this to 0.676 without ground-truth poses. Results on EuRoC and TUM RGB-D show that VIDAR is a practical route to metric dense monocular reconstruction.

## 1. 核心问题与整体含义（研究动机与背景）

- **研究动机**：移动机器人需要密集几何地图用于导航、避障、场景理解等任务。单目几何基础模型（如 Depth Anything 3）虽然能够推断密集的局部几何，但**缺乏稳定且物理可解释的度量尺度**，导致重建结果在不同序列中出现尺度不确定性，无法直接用于实际机器人系统中的度量建模。另一方面，经典的视觉惯性里程计（VIO）能提供高效的位姿估计和度量轨迹，但本身不产出密集几何。  
- **核心问题**：如何将两者的优势结合起来——让基础模型提供局部密集几何，同时利用 VIO 提供度量尺度和稳定世界坐标系，从而得到**度量且致密的全局场景重建**。  
- **整体含义**：本文提出 **VIDAR**，一种混合框架：SVO+IMU 作为轻量级度量锚点，Depth Anything 3 作为密集几何来源。它不是简单取代某一模块，而是证明两者互补的价值，为低成本机器人实现度量密集重建提供了实用路线。

## 2. 方法论：核心思想、关键技术细节、流程

### 2.1 核心思想
- **分工明确**：VIO 前端提供度量轨迹（尺度、位姿、全局坐标系）；DA3 提供密集局部几何（深度图）。最终重建继承 VIO 的度量放置与 DA3 的局部细节。
- **两种耦合策略**：
  1. **位姿条件化（Pose-Conditioned）**：在 DA3 推理阶段直接注入 VIO 的相机外参（`ext = T_CW`），强制 DA3 在当前度量框架下预测深度，之后通过 Umeyama 比例对齐进一步微调 DA3 预测的相机中心与 VIO 中心之间的尺度。
  2. **解耦混合（Decoupled Hybrid）**：DA3 先在自有无位姿框架下独立重建，保持其局部几何先验；之后将 DA3 的相机中心与 VIO 的相机中心进行单个相似变换（Umeyama 求解 `(s, R, t)`），将整个 DA3 点云对齐到 VIO 的度量框架。此变体避免了修改 DA3 内部推理，且更适合短或低漂移序列。

### 2.2 关键技术细节
- **视觉惯性几何锚定**：  
  对像素 `(u,v)`，给定 DA3 预测深度 `D_i(u,v)` 和相机内参 `K`，局部点 `x_i = D_i(u,v) K^{-1}[u,v,1]^T`，用 VIO 位姿 `T_w^i` 变换到世界坐标系 `x_w = T_w^i x_i`。
- **Umeyama 相似变换**：  
  对于重叠点集 `P_s, P_t`，计算 `s = σ_t/σ_s`（标准差比），通过 SVD 得到旋转矩阵和位移，得到 `S = (s, R, t)`，用于对齐 DA3 块与 VIO 轨迹。
- **鲁棒性防护**：  
  尺度超出 `[0.85, 1.20]` 时钳制为 1；秩不足时退化为单位变换；在累积图优化中加入闭环约束（基于图像描述符余弦相似度），并执行 Sim(3) 位姿图优化。

### 2.3 算法流程（文字说明）
1. 输入 RGB 序列（及同步 IMU 数据）；
2. SVO+IMU 前端估计度量轨迹 `{T_w^i}`；
3. 对每个图像帧或序列块，DA3 生成密集深度图；
4. 两种耦合方式之一对齐深度图到 VIO 坐标系；
5. 通过重叠块的 Umeyama 对齐合并相邻块，形成全局点云；
6. （可选）闭环检测与 Sim(3) 位姿图优化修正累积漂移。

## 6. 主要结论与发现

1. **VIO 是有效的度量锚点**：SVO+IMU 在 EuRoC 上 Sim(3)-aligned RMSE 降至 0.125 m（单目 SVO 为 0.898 m），表明其能提供稳定的度量轨迹。
2. **VIO 注入大幅改善尺度误差**：位姿条件 DA3 的尺度误差 `|ln s|` 从 0.458 降至 0.009，F@0.10 从 0.405 提升至 0.463（接近 GT 位姿的 0.504）。
3. **解耦混合策略最优**：在不使用真实位姿的情况下，解耦混合 VIDAR 达到平均 F@0.10 = 0.676，说明保留 DA3 原始局部几何+仅借用 VIO 度量框架，效果优于直接替换位姿。
4. **DA3 在 RGB-only 环境中弥补单目 VIO 弱点**：TUM 上 DA3 轨迹 RMSE 仅 0.031 m，远优于单目 SVO 的 0.429 m。
