## 摘要
跨图像比较推理对视觉语言模型（VLM）仍是一项挑战，尤其在正确预测需要细粒度属性定位和全局一致推理时。我们提出了CoRe，一个针对该问题的统一框架。CoRe包括：（i）CoRe-20K，一个大规模基于三元组的训练集，通过多专家协作流程从结构化视觉元数据自动构建，涵盖计数、深度、距离和空间关系；（ii）TriSR，一个结构化奖励框架，在GRPO优化下联合监督属性定位、判断对齐和三重一致性；（iii）CoRe-Bench，首个专门用于细粒度跨图像比较推理的基准。实验表明，CoRe在CoRe-Bench上显著优于现有VLM，同时在标准多模态基准上保持竞争力，部分准确率比最强基线高出28.2个百分点。

## Abstract
Cross-image comparative reasoning remains challenging for vision-language models (VLMs), especially when correct prediction requires fine-grained attribute grounding and globally consistent reasoning. We present CoRe, a unified framework for this problem. CoRe includes: (i) CoRe-20K, a large-scale triplet-based training set automatically constructed from structured visual metadata through a multi-expert collaborative pipeline, covering counting, depth, distance, and spatial relations; (ii) TriSR, a structured reward framework that jointly supervises attribute grounding, judgment alignment, and triplet consistency under GRPO optimization; and (iii) CoRe-Bench, the first benchmark dedicated to fine-grained cross-image comparative reasoning. Experiments show that CoRe substantially outperforms existing VLMs on CoRe-Bench while remaining competitive on standard multimodal benchmarks, achieving a 28.2-point gain in partial accuracy over the strongest baseline.

## 1. 核心问题与整体含义
- **研究动机**：视觉语言模型（VLM）在单图像理解任务中表现优异，但在需要跨图像精细比较（如物体数量、深度、距离、空间关系）的任务中系统性失效。现有基准多聚焦于单图像感知或宏观多图像语义理解（如时间排序、叙事理解），缺乏对细粒度跨图像比较推理的专门评估和训练数据。
- **核心挑战**：（1）结果导向的优化（仅监督最终答案正确性）会导致模型通过捷径或偶然猜测得出正确回答，但推理链错误或缺乏全局一致性（实验显示高达17.1%的正确答案伴随无效推理）；（2）使用LLM作为评判者存在与人类判断的偏差，不适合结构化推理验证。
- **整体含义**：跨图像比较推理是一项亟待提升的独特能力，需要可验证的中间结构监督。

## 2. 方法论
### 2.1 核心思想
提出 **CoRe 统一框架**，包含：
- **CoRe-20K**：基于结构化元数据自动构建的大规模三元组（triplet）训练集（覆盖计数、深度、距离、空间关系四类，每类5000三元组，共60000 QA对）。
- **TriSR**：结构化奖励框架，在GRPO优化下联合监督**属性定位（Attribute Grounding）**、**判断对齐（Judgment Alignment）**和**三元组一致性（Triplet Consistency）**。
- **CoRe-Bench**：首个专门用于细粒度跨图像比较推理的基准（4000 held-out三元组）。

### 2.2 关键技术
1. **元数据驱动三元组构建**（多专家协作流程）：
   - **Metric Extraction Expert**：从源数据集中提取任务相关度量（例如从注释中读取计数、深度中值、欧氏距离、空间关系标签）。
   - **Quality Control Expert**：过滤不可靠或琐碎三元组（设定差值阈值满足[\theta_min, \theta_max]）。
   - **Question Generation Expert**：基于模板库生成问题，随机化选项位置消除偏差。

2. **TriSR 结构化奖励**：
   - **Attribute Grounding Reward (R_ag)**：鼓励模型正确估计每幅图像的属性，采用指数度量误差衰减。
   - **Judgment Alignment Reward (R_ja)**：同时检查推理链中的判断是否匹配真实答案和最终答案。
   - **Triplet Consistency Reward (R_tc)**：通过逻辑一致性检验（如传递性）确保三元组比较自洽。
   - **Composite Reward**: R_total = R_task + λ_ag R_ag + λ_ja R_ja + λ_tc R_tc，λ分别设为0.2、0.2、0.3。

3. **GRPO优化**：对每个查询采样G个输出，计算群体归一化优势，结合KL正则化更新策略。

- CoRe在CoRe-Bench上**整体准确率（Overall）** 达22.2%（基线最高6.6%），**部分准确率（Partial）** 达54.0%（基线最高25.8%），提升28.2个百分点。

- 在**通用基准**上，CoRe-4B在5/9个基准上达到开源模型最优，且不出现灾难性遗忘。
- **结论**：利用可验证的中间结构（属性定位、判断一致性、三元组一致性）进行监督是提升跨图像比较推理的有效方向。

## 7. 优点
1. **数据自动构建**：采用多专家协作管道从结构化元数据生成可比数据，避免人工标注或VLM伪标签误差，保证标签可验证性。
2. **结构化奖励设计**：TriSR将复杂推理评估分解为三个可计算、可验证的指标，监督既细致又可靠，避免了LLM-as-judge的主观偏差。
3. **首个专用基准**：CoRe-Bench系统覆盖四种比较维度，填补了细粒度跨图像比较推理评估空白。
4. **训练与泛化兼顾**：在专用任务上提升显著，同时不牺牲通用能力（甚至在多个基准上超越更大模型），体现了方法的实用价值。
5. **分析深入**：通过推理链分类和置信区间揭露了结果监督的缺陷，验证了结构化奖励的必要性。

## 8. 不足与局限
1. **任务覆盖有限**：仅涵盖计数、深度、距离、空间关系四种属性，缺乏其他可能比较维度（如颜色、纹理、运动等）。
