## 摘要
增量式场景重建对于实际应用至关重要。尽管3D高斯泼溅展现出巨大潜力，但现有方法大多需要将优化后的高斯体离线转换为中间隐式场以提取显式网格，这阻碍了与下游任务的无缝集成。为解决这一限制，我们提出了一种新颖的在线框架，通过直接对密集的几何高斯表示进行三角化，增量地重建和更新高保真显式网格，同时支持高质量渲染和增量式表面重建。此外，我们提出了一种直接网格化算法，能够高效地从高斯集合中提取和更新网格。为确保网格精度，我们实施了一种基于平面的牵引约束，动态地将3D高斯基元对齐到近似的局部表面。最后，我们的框架通过动态冻结完全优化的历史区域，在长序列处理过程中显著减少了内存和计算开销。在公开数据集上的实验表明，我们的方法在渲染质量和重建精度上均优于传统基于高斯的方法。

## Abstract
Incremental scene reconstruction is essential for real-world applications. Although 3D Gaussian Splatting shows strong potential, most existing approaches require offline conversion of the optimized Gaussians into an intermediate implicit field for explicit mesh extraction, which hinders seamless integration with downstream tasks. To address this limitation, we propose a novel online framework that incrementally reconstructs and updates high-fidelity explicit meshes by directly triangulating a dense geometric Gaussian representation, which supports both high-quality rendering and incremental surface reconstruction. Moreover, we present a direct meshing algorithm that efficiently extracts and updates the mesh from the Gaussian set. To ensure mesh accuracy, we enforce a plane-based pulling constraint that dynamically aligns 3D Gaussian primitives to the approximated local surface. Furthermore, our framework significantly reduces memory and computational overhead during long-sequence processing by dynamically freezing fully optimized historical regions. Experiments on public datasets demonstrate that our method outperforms conventional Gaussian-based methods on both rendering quality and reconstruction accuracy.

## 2. 方法论

### 核心思想
- 将3D高斯体视为**平面椭圆曲面元（Planar Elliptical Surfels）**，通过强制高斯体的第三尺度分量趋近于零（平面约束）和高不透明度（接近1）使其贴合局部表面。
- 设计**直接网格化算法**，对几何高斯集合进行快速三角化，并支持增量式局部重新网格化与冻结策略。
- 引入**平面拉拽约束**，利用从RGB-D流得到的定向点云，将高斯中心拉向局部平面，使其与真实表面对齐。

### 关键技术细节
1. **密集几何高斯表示**：
   - 每个高斯体包含均值 \(\mu_i\)、不透明度 \(\alpha_i\)、颜色 \(\mathbf{c}_i\)、尺度向量 \(\mathbf{s}_i\)（第三分量强制为 \(10^{-6}\)）、旋转矩阵 \(\mathbf{R}_i\)（第三列对齐法线）。
   - 初始化时，高斯均值继承自定向点云，切向尺度由局部点间距决定（k=4最近邻）。

2. **几何感知可微分渲染**：
   - RGB渲染使用标准alpha混合；深度渲染通过计算视线与最前端不透明高斯体的交点得到，增强了几何精度。

3. **在线映射与优化**：
   - **高斯添加**：在深度与颜色差异大的区域补充不透明或半透明高斯体。
   - **高斯剪枝**：移除不参与深度渲染的不透明高斯和贡献小的透明高斯。
   - **损失函数**：
     - 颜色损失 \(L_{\text{color}}\)、深度损失 \(L_{\text{depth}}\)；
     - 平面拉拽损失 \(L_{\text{plane}} = \sum |\mathbf{n}_i^\top (\mu_i - \mathbf{p}_i)|\)；
     - 法线一致性损失 \(L_n = 1 - |\mathbf{n}_g^\top \mathbf{n}_i|\)；
     - 稀疏损失 \(L_{\text{sparse}}\) 鼓励高斯要么透明要么不透明。
   - 总损失：\(L = \lambda_c L_{\text{color}} + \lambda_d L_{\text{depth}} + \lambda_p L_{\text{plane}} + \lambda_n L_n + \lambda_s L_{\text{sparse}}\)。

4. **高斯三角化**：
   - 首先筛选出**几何高斯集合**：满足不透明度 > 0.9 且投影深度与真实深度差异 < \(\tau=0.001\)。
   - **快速三角化**：对每个几何高斯，搜索自适应半径内的邻居，施加互相可见性和法线一致性约束，筛选后通过加权融合细化中心位置，然后投影到切平面并按角度贪婪连接形成三角形。
   - **局部重新网格化与冻结**：对新增区域进行边分裂、边坍缩、边翻转和拉普拉斯平滑，消除拓扑冲突；当区域观测次数超过阈值 \(N_{\text{obs}}=10\) 且损失稳定低于 \(\epsilon_{\text{gs}}=0.1\) 时，冻结该区域的网格和高斯，不再参与优化。
