## 摘要
现有的3D场景基础大型语言模型（3D-LLMs）专注于回答基于简化的单房间3D场景的问题，缺乏对包含多个互联房间和多样物体类别的真实世界家居环境进行推理的能力。我们提出了CAIRN，一种用于多房间3D场景理解的拓扑感知3D-LLM。CAIRN将Transformer注意力与场景层级对齐，使模型对物体级关系和房间级连接具有显式感知。它通过图神经网络用房间局部关系上下文丰富物体标记，引入可学习的房间标记用于房间级抽象，并应用具有几何偏置的分层注意力掩码，根据场景拓扑路由信息。CAIRN基于CAIRN-MR开发，这是我们针对多房间3D场景理解在HM3D上引入的基准，涵盖定位、描述和四个问答任务，逐步从房间内感知评估到跨房间推理。实验表明，CAIRN在CAIRN-MR的所有任务上大幅优于先前的3D-LLMs，同时在五个单房间基准上保持竞争力。

## Abstract
Existing 3D scene-grounded Large Language Models (3D-LLMs) focus on answering questions grounded in simplified single-room 3D scenes, lacking the ability to reason over real-world household environments containing multiple interconnected rooms and diverse object categories. We introduce CAIRN, a topology-aware 3D-LLM for multi-room 3D scene understanding. CAIRN aligns transformer attention with scene hierarchy, giving the model explicit awareness of object-level relations and room-level connectivity. It enriches object tokens with room-local relational context via a graph neural network, introduces learned room tokens for room-level abstraction, and applies a hierarchical attention mask with geometric bias to route information according to scene topology. CAIRN is developed on CAIRN-MR, a benchmark we introduce on HM3D for multi-room 3D scene understanding, covering grounding, captioning, and four question-answering tasks that progressively evaluate from intra-room perception to cross-room reasoning. Experiments show that CAIRN outperforms prior 3D-LLMs by a large margin across all CAIRN-MR tasks while remaining competitive on five single-room benchmarks.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：现有3D场景基础的大型语言模型（3D-LLMs）主要关注简化的单房间场景，将3D场景表示为平坦的对象令牌序列，缺乏对房间级结构的建模。真实家居环境包含多个互联房间和数百个物体，需要进行跨房间推理（如物体检索、跨空间比较）。
- **核心问题**：如何使3D-LLM具备对多房间3D场景的层次化结构（房间级连接、物体间关系）的显式理解，从而支持房间内感知、房间定位、跨房间比较等复杂推理任务。
- **研究动机**：现有方法将单房间范式扩展到多房间时面临两大挑战：
  1. 平坦的自注意力机制不适合多房间场景（跨房间交互稀疏，主要交互发生在房间内部）；
  2. 多房间场景蕴含几何和拓扑先验（相对位姿、距离、房间邻接），但标准自注意力未利用这些信息。

## 2. 论文提出的方法论

### 核心思想
- **CAIRN**：一个拓扑感知的3D-LLM，通过显式建模多房间3D场景的层次结构和拓扑，将信息流与场景拓扑对齐。
- **总体流程**：输入3D场景 → 构建层次化场景图 → 图编码为场景令牌（含物体令牌和房间令牌） → 通过带几何偏置的分层掩码注意力送入LLM进行推理。

### 关键技术细节

1. **层次化场景图构建**：
   - 两层图：下层为物体级结构（节点为物体实例，边为同一房间内K个最近邻对象），上层为房间级拓扑（节点为房间，边为相邻房间），跨层边连接物体到所属房间。
   - 物体实例通过Mask3D分割获得，房间通过鸟瞰图占据分区获得。

2. **层次化场景图令牌化**：
   - **物体级令牌**：每个物体Oi映射为4个令牌：标识符令牌（文本ID）、3D几何令牌（Uni3D特征投影）、2D外观令牌（DINOv2多视图图像特征投影）、图令牌（通过Lg层消息传递聚合邻居信息后投影）。
   - **房间级令牌**：每个房间Rr引入Kr个可学习查询，通过交叉注意力聚合房间内物体令牌，生成房间令牌，用于跨房间通信。

3. **分层掩码注意力与几何偏置**：
   - **分层掩码MH**：根据场景图定义令牌间可见性，仅允许物体-物体系（同房间）、物体-房间系、房间-房间系（相邻房间）的交互。由此实现层次化路由：物体只与同房间物体交互，跨房间信息通过房间令牌中继。
   - **几何偏置Bθ**：针对每种交互类型（OO、OR、RR），学习一个MLP，输入空间描述符（如相对位置、距离、物体-房间位置、房间相对布局）输出偏移量加到注意力logits中。偏置仅在前Lb层应用，后续层仅保留掩码带来的稀疏性。
   - 最终注意力：Attn(X) = Softmax((QK^T/√d) + MH + Bθ) V。
   - 效果：将注意力变为场景拓扑的结构化算子，减少计算复杂度从O(N²)到O(∑|O(r)|²)。

### 公式说明（文字描述）
- 场景图：GH = ( (VO, EOO), EOR, (VR, ERR) )。
- 物体令牌：T_Oi = [tid_i, t3D_i, t2D_i, tG_i]。
- 房间令牌：tRr,k = WR CrossAttn(qr,k, {ti})。
- 注意力加性项：分层掩码MH决定哪些对可以交互，几何偏置Bθ调节交互强度。

## 6. 论文的主要结论与发现

1. **拓扑感知显著提升多房间理解**：CAIRN在CAIRN-MR所有任务上大幅超越基线，定位提升5.4 Acc@0.25，描述提升14.9 CIDEr（~19%相对提升），比较任务提升6.7 EM。
2. **增益随房间级推理需求增加而增大**：跨房比较和存在性验证增益最大，计数中等，识别任务增益最小（因为识别更多依赖局部物体识别）。
3. **单房间性能不受损**：CAIRN在5个单房间基准上与最强方法持平或略优，证明房间级组件与单房间场景兼容。
4. **分层掩码是关键驱动**：消融显示去掉分层掩码导致最大性能下降（比较任务-5.5 EM，描述-6.7 CIDEr），几何偏置和结构化表示提供互补增益。
5. **基准质量可靠**：人工性能高（86.8%-94.5%），基线差异显著，任务需要真实空间推理。

## 8. 不足与局限
