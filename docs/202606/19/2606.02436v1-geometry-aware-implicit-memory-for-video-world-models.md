## 摘要
视频世界模型旨在模拟可控的视觉环境，但长程推演取决于模型在观测离开其原生上下文窗口后记忆的内容。显式记忆保留帧或在线3D重建，可能面临启发式检索错误、冗余外观存储或重建伪影。隐式记忆将历史压缩为紧凑状态，但现有设计未明确约束编码跨视图场景几何。我们提出GIM-World，一种用于视频世界模型的几何感知隐式记忆框架。轻量级Transformer编码器将变长历史压缩为固定大小的记忆令牌，一个可查询相机的几何头在训练期间将冻结基础模型的3D场景结构蒸馏到记忆中，一个信息引导的剪枝规则在历史增长时保持编码成本有界。推理时丢弃几何教师，留下轻量级记忆模块。在MIND上的实验表明，GIM-World比显式记忆和隐式记忆基线更好地保持长程几何和视觉一致性。

## Abstract
Video world models aim to simulate controllable visual environments, but long-horizon rollouts depend on what the model remembers after observations leave its native context window. Explicit memories retain frames or online 3D reconstructions, which can suffer from heuristic retrieval errors, redundant appearance storage, or reconstruction artifacts. Implicit memories compress history into a compact state, but existing designs are not explicitly constrained to encode cross-view scene geometry. We propose GIM-World, a geometry-aware implicit memory framework for video world models. A lightweight transformer encoder compresses variable-length history into fixed-size memory tokens, a camera-queryable geometry head distills 3D scene structure from a frozen foundation model into the memory during training, and an information-guided pruning rule keeps encoding cost bounded as history grows. The geometry teacher is discarded at inference, leaving a lightweight memory module. Experiments on MIND show that GIM-World better preserves long-horizon geometric and visual consistency than both explicit- and implicit-memory baselines.

## 2. 论文提出的方法论
### 核心思想
- **隐式记忆编码器**：将变长的历史帧序列（含相机位姿）通过一个轻量级Transformer编码器压缩为**固定大小**的记忆令牌集合（`Nm`个）。编码器只有两个block，使用紧凑自注意力（Compact Self-Attention）和全分辨率前馈网络，推理开销极低（小于DiT解码时间的0.3%）。
- **相机可查询几何监督头**：训练时，从历史中随机采样一帧，利用该帧的相机位姿生成射线查询（ray queries），通过跨注意力从记忆中检索几何信息，并与该帧对应的**VGGT特征图**计算逐块余弦损失 `L_geo`。该几何头仅在训练时使用，推理时丢弃。
- **信息引导的剪枝规则**：为控制编码成本，当历史帧数增长时，采用基于高斯过程（GP）的互信息准则对历史进行子集选择（保留最多K帧），通过贪心算法最大化所选子集与剩余帧之间的互信息。核函数综合相机位置、朝向和时间信息。

### 关键技术细节
- 记忆令牌初始化为一组可学习的查询（`Q0`），与历史帧令牌拼接后输入编码器。
- 几何监督使用**射线图（Ray Map）**，将相机位姿转换为每像素的射线（原点+方向），通过MLP嵌入与共享的2D位置编码相加作为查询。
- 总训练损失：`L = L_FM + λ L_geo`，其中 `L_FM`为流匹配损失（生成未来帧），`λ=0.05`。整个框架端到端单阶段训练。
- 推理时，仅保留记忆编码器和DiT生成器，几何头与VGGT教师模型均被丢弃。

## 7. 优点
- **方法优雅高效**：隐式记忆固定大小，训练时加入几何蒸馏但在推理时完全丢弃额外模型，无任何推理开销增加。
- **几何蒸馏设计巧妙**：通过相机查询解决记忆令牌与输入无对应关系的问题，避免了直接特征对齐的虚假约束。
- **理论驱动的剪枝**：采用GP互信息准则（源于传感器放置理论），相比朴素启发式更优，且可解释。
- **端到端单阶段训练**：无需预训练或分阶段蒸馏，流程简洁。
- **大量消融实验**：证实了各组件（几何监督、剪枝、注意力压缩）的有效性，验证充分。

## 8. 不足与局限
- **数据集单一**：仅在合成数据集MIND上评估，未在真实世界视频（如机器人、驾驶）或开放域视频上进行测试，可能高估在复杂动态场景中的表现。
- **依赖VGGT特征**：教师模型可能引入自身偏差（例如对弱纹理、动态物体不鲁棒），且蒸馏损失使用余弦相似度，可能不足以捕捉复杂几何细节。
- **剪枝控制参数需人工设定**：GP核函数的带宽（σp, σr, σt）需要根据场景手动调整，缺乏自适应机制。
- **隐式记忆容量固定**：未实验不同`Nm`大小的影响，记忆容量与历史复杂度的匹配关系未被探讨。
- **对比基线不够全面**：隐式记忆仅复现VideoSSM（SSM），缺少与其他近期隐式工作（如StateSpaceDiffuser, Infinite-World）的比较。显式记忆中的3D重建类方法（如Gen3C, VMEM）也未直接对比（但注明了MIND benchmark没有对应公开结果）。
- **动态对象挑战**：论文框架主要关注静态场景几何一致性，对动态对象（如NPC、角色动作）的处理能力未深入分析（第三人称定量结果中角色保持较好，但可能受益于MIND动画的规律性）。
