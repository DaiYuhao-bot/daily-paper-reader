## 摘要
可微分渲染已成为三维重建和新视图合成的强大方法。最先进的可微分渲染方法将多种自定义的三维几何和外观表示与专门的渲染器相结合。然而，计算机图形学中的大多数下游任务依赖于三维网格。虽然先前的工作尝试了使用网格表示的可微分渲染，但这些方法仅限于以物体为中心的场景，无法重建大规模、无边界场景。在这项工作中，我们提出Meshtryoshka，一种新颖的网格可微分渲染框架，它将现成的三角形光栅化器与由嵌套网格壳组成的三维表示相结合，类似于套娃。在每次前向传递中，通过等值面提取从三维符号距离函数中重新提取网格壳，每个顶点的不透明度计算为符号距离的函数。然后每个网格壳独立光栅化，最终图像通过alpha合成创建。关键的是，网格顶点位置仅通过梯度间接更新，梯度通过不透明度值流入符号距离函数，因此我们的方法与现成的网格渲染器兼容，这些渲染器不需要对顶点位置可微分。在以物体为中心的场景中，我们的方法与基于表面的可微分渲染技术相比表现有竞争力。我们的可微分网格渲染方法可扩展到无边界的真实三维场景，其中它产生的高质量新视图合成结果接近最先进的非网格方法。我们的方法表明，可能无需依赖专门的渲染器，仅使用计算机图形学工具箱中的常规工具就能解决可微分渲染问题。

## Abstract
Differentiable rendering has emerged as a powerful approach for 3D reconstruction and novel view synthesis. State-of-the-art differentiable rendering methods combine a variety of custom representations of 3D geometry and appearance with specialized renderers. However, most downstream tasks in computer graphics rely on 3D meshes. While prior work has attempted differentiable rendering with mesh representations, these approaches are limited to object-centric scenes and fail to reconstruct large-scale, unbounded scenes. In this work, we introduce Meshtryoshka, a novel mesh differentiable rendering framework that combines an off-the-shelf triangle rasterizer with a 3D representation that consists of nested mesh shells which resemble a matryoshka doll. In every forward pass, the mesh shells are extracted anew from a 3D signed distance function via iso-surface extraction, and the opacities for each vertex are computed as a function of signed distance. Each mesh shell is then rasterized independently, and the final image is created via alpha compositing. Crucially, mesh vertex positions are updated only indirectly via gradients that flow through the opacity values into the signed distance function, and hence, our method is compatible with off-the-shelf mesh renderers that need not be differentiable with respect to vertex positions. On object-centric scenes, our method performs competitively with surface-based differentiable rendering techniques. Our differentiable mesh rendering method scales to unbounded, real-world 3D scenes, where it yields high-quality novel view synthesis results approaching those of state-of-the-art, non-mesh methods. Our method suggests that it may be possible to solve the differentiable rendering problem without relying on specialized renderers, only using conventional tools from the computer graphics toolbox.

### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有的可微分渲染方法大多依赖自定义的3D表示（如神经辐射场NeRF、3D高斯溅射）和专门的渲染器，但计算机图形学的主流工作流以三角形网格为核心（可移植、硬件加速、易编辑）。先前基于网格的可微分渲染方法（如DMTet、FlexiCubes）仅能处理物体中心的简单场景，且需前景背景掩码，无法扩展到无边界的真实世界场景。
- **动机**：是否有途径仅使用传统计算机图形学工具（如现成的三角形光栅化器）来实现高质量的可微分渲染，从而直接优化网格表示？本文提出的Meshtryoshka正是尝试回答这一问题。

### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将场景表示为一组从符号距离函数（SDF）中提取的嵌套网格壳（类似套娃），每个壳用现成光栅化器独立渲染，然后通过alpha合成得到最终图像。关键创新是**顶点位置不直接接受梯度**，而是通过不透明度（opacity）梯度间接更新SDF参数，从而兼容非可微分的标准光栅化器。
- **关键技术细节**：
  1. **场景参数化与网格提取**：在稀疏网格上存储SDF值和球谐系数，通过可微分的Marching Cubes提取多个等值面（对应不同符号距离水平），每个等值面形成一个三角形网格壳。
  2. **非可微分光栅化与延迟渲染**：用现成光栅化器对每个壳独立渲染，获得像素级的三角形ID和重心坐标；然后通过可微分插值得到像素级SDF值和颜色；再通过SDF到不透明度的映射（类似NeuS公式）计算alpha值；最后按层进行alpha合成。
  3. **稀疏性与粗到细优化**：使用比特掩码表示活动网格（只存储边界附近的体素），结合稀疏Marching Cubes，大幅降低内存和计算。每800步进行一次剪枝和细分（从初始64³逐步升至1024³）。
  4. **无边界的场景处理**：物体中心场景用常规立方网格；真实世界场景分为前景（轴对齐立方体）和背景（六个截锥体），背景的顶点距离呈指数分布，使体素尺寸均匀增长，适配Marching Cubes。
  5. **正则化**：使用拉普拉斯平滑损失（鼓励SDF值局部均匀）和L1惩罚（使未观测区域SDF快速趋于±∞，从而修剪几何）；对球谐的低阶部分降低学习率并加L2惩罚，防止局部极小。
- **公式与算法流程**：
  - 等值面对应符号距离`ℓ`，通过插值权重`t = (d_j - ℓ)/(d_j - d_i)`确定顶点位置和属性。
  - 不透明度由`α_i = (1 - T_i)`（最内层）或`(T_{i-1} - T_i)/T_{i-1}`推导，其中`T_i = 1/(1+exp(-d_i))`。
  - 最终像素颜色为逐层alpha合成：`C = Σ (α_i * c_i * Π_{j<i} (1-α_j))`。

### 6. 主要结论与发现
- 在物体中心场景（NeRF-Synthetic）上，Meshtryoshka与nvdiffrec（FlexiCubes）性能相当（PSNR 29.36 vs 29.22），无需前景掩码，且优于nvdiffrec+DMTet和Volumetric Surfaces。
- 在无边界的Mip-NeRF 360场景上，首次证明基于网格的可微分渲染可行，PSNR达到24.34 dB，虽落后于Zip-NeRF（28.54）和3D Gaussian Splatting（27.21），但已接近竞争水平，且是**唯一成功重建该数据集的网格可微渲染方法**。
- **关键发现**：仅使用现成三角形光栅化器，结合嵌套壳结构和间接梯度更新，即可实现高质量的3D重建和新视图合成，表明可微分渲染不一定依赖专用渲染器。

### 7. 优点
- **无需专用可微渲染器**：完全兼容标准硬件加速光栅化器，便于移植和工程集成。
- **直接输出网格**：优化过程中直接获得三角形网格，无需后处理“烘焙”步骤，质量与最终渲染一致。
- **可处理无边界场景**：通过截锥体和指数分布顶点融入标准的Marching Cubes框架，突破了先前网格方法的空间限制。
- **稀疏与粗到细优化**：显著降低内存和计算量，使高分辨率网格训练可行。
- **无需前景掩码**：对比nvdiffrec等方法更灵活。
