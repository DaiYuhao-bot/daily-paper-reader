## 摘要
移动操作机器人需要当前可查询、语义丰富且能在边缘计算约束下使用的世界模型。本技术报告提出KRVF，一种面向边缘移动操作的源感知语义体素世界表示。与主要优化全局几何保真度的以重建为中心的建图流水线不同，KRVF将局部世界状态表示为面向任务的体素，编码占据、颜色、语义证据、时间新鲜度和证据来源。该表示将测量得到的占据信息与语义先验假设分离，从而实现能够感知深度故障的物体推理，而不会静默地破坏持久几何结构。KRVF还通过渲染地图先验深度进行修复，闭合了建图与感知之间的反馈环路，并公开了面向任务级别的语义物体和抓取候选的查询算子。报告形式化定义了KRVF表示，并记录了将在线RGB-D观测转换为面向任务的机器人记忆的ROS 2实现。

## Abstract
Mobile manipulators need world models that are current, queryable, semantically meaningful, and usable under edge-compute constraints. This technical report presents KRVF, a source-aware semantic voxel world representation for edge mobile manipulation. Unlike reconstruction-centric mapping pipelines that primarily optimize global geometric fidelity, KRVF represents local world state as task-oriented voxels that encode occupancy, color, semantic evidence, temporal freshness, and evidence source. The representation separates measured occupancy from semantic-prior hypotheses, enabling depth-failure-aware object reasoning without silently corrupting persistent geometry. KRVF also closes a feedback loop between mapping and sensing by rendering map-prior depth for repair, and exposes task-level query operators for semantic objects and grasp candidates. The report formalizes the KRVF representation and documents a ROS 2 implementation that turns online RGB-D observations into a task-facing robot memory.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：边缘移动操作机器人在有限计算资源下，需要一种世界模型，能够提供**低延迟、可查询、语义丰富、对深度传感器失效鲁棒**的表示，以支撑任务级决策（如物体在哪里、证据是否可靠、抓取候选在何处）。
- **背景**：现有方法（如OctoMap占用建图、TSDF稠密融合、RTAB-Map RGB-D SLAM、神经辐射场等）主要优化**全局几何保真度或重建质量**，并未针对**边缘操作所需的源感知、假设分离、临时新鲜度和任务级查询**进行设计。特别是深度传感器在透明、反光、暗面、薄物等场景下失效时，纯几何建图会错误地将缺失深度视为自由空间，导致物体被“削除”。
- **整体含义**：KRVF将地图视为**源感知、可查询的机器人任务记忆**，而非仅几何重建。核心思想是机器人应跟踪**为何相信某区域被占据、该信念是否仍有效、该结构是否能支持任务**。

## 2. 方法的核心思想、关键技术细节、算法流程
### 核心思想
- **源感知语义体素场**：每个体素编码占据对数几率、颜色、语义证据、观测置信度、时间新鲜度、证据类型（观测/修复/假设/不确定）。
- **观测-假设分离**：将测量占据（\(K_\text{obs}\)）与语义先验假设（\(K_\text{hyp}\)）明确分开。语义假设不自动合并到持久几何中，有生命周期，可过期或被后续有效深度证据提升。
- **地图先验反馈回路**：将当前体素地图渲染为深度先验，用于修复损坏深度，修复后的深度以置信度方式融合回地图。
- **任务级查询接口**：提供语义物体查询和抓取候选查询（顶部、腰部、侧面、后备），基于物体稳定性和几何启发式排序。

### 关键技术细节
- **源加权融合**：占据更新 \( \Delta l_i = q_t \cdot L_\text{hit}(z_t, x_t, v_i) \)，其中 \( q_t \) 是观测源置信度（原始深度、修复深度、语义假设具有不同置信度）。
- **语义门控逆传感器模型**：\( \Delta l_i = \alpha_i L_\text{phys} + (1 - \alpha_i) L_\text{sem} \)，当物理深度可靠时主导，不可靠时语义先验创建假设。
- **动态清除**：占据体素累计连续缺失观测后对数衰减，衰减速率取决于局部支撑。
- **颜色稳定**：持续/实时双色估计，多帧融合避免单帧噪声覆盖。
- **候选到稳定提升**：候选体素经多次支持后晋级稳定状态。

### 算法流程（文字说明）
1. 输入同步RGB-D图像、相机标定、位姿变换。
2. 深度有效点反投影到三维，变换到世界系，插入分块稀疏体素地图。
3. 融合占据、颜色、置信度和时间信息，并执行保守射线投射记录自由空间。
4. 语义桥节点处理2D检测（如YOLO）：采样检测框内深度点，过滤不稳定边缘，将标签更新到已有占据体素；若深度缺失，在假设层创建带界限的假设体素。
5. 可选：将当前体素地图渲染为深度先验，输入深度修复模块，修复后深度以置信度融合。
6. 下游模块通过服务/动作接口查询语义物体和抓取候选。
