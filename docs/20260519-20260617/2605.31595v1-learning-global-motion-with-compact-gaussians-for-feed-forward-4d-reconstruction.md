## 摘要
从单目视频进行动态场景重建仍然是计算机视觉中的一个基本挑战。现有的前馈方法逐帧逐像素预测3D高斯，存在重复高斯和视角依赖偏差的问题，阻碍了场景运动的高效学习。我们提出C4G，一种基于紧凑的可学习时间戳条件高斯查询标记的前馈式4D重建框架。每个标记在整个时间上下文中聚合对应特征，并解码出一个3D高斯，其位置由目标时间戳调制，从而实现全局一致的运动建模，无需逐场景优化。为捕捉精细细节，我们进一步引入基于视频扩散模型的渲染增强模块。由于我们的框架有效地将特征聚合到高斯中，我们将此能力扩展到特征提升，生成支持点跟踪和动态场景理解的4D特征场。C4G在显著减少高斯数量的情况下，无需相机姿态，即可实现强时效的新视角合成性能，同时展现出更强的运动建模能力和对较大时间间隔的鲁棒性。

## Abstract
Dynamic scene reconstruction from monocular video remains a fundamental challenge in computer vision. Existing feed-forward methods predict 3D Gaussians pixel-wise for each frame, suffering from duplicated Gaussians and view-dependent biases that hinder effective learning of scene motion. We present C4G, a feed-forward 4D reconstruction framework built upon a compact set of timestamp-conditioned learnable Gaussian query tokens. Each token aggregates corresponding features across the full temporal context and decodes a 3D Gaussian whose position is modulated by the target timestamp, enabling globally coherent motion modeling without per-scene optimization. To capture fine-grained details, we further introduce a video diffusion model-based rendering enhancement module. Since our framework effectively aggregates features into Gaussians, we extend this capability to feature lifting, producing a 4D feature field that supports point tracking and dynamic scene understanding. C4G achieves strong novel-view synthesis performance using significantly fewer Gaussians and without requiring camera poses, while exhibiting stronger motion modeling and robustness to large temporal gaps.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：从单目视频进行动态场景（4D）重建是一个基础且具有挑战性的计算机视觉任务。现有最先进的前馈式（feed-forward）方法依赖于**逐像素预测**3D高斯，这导致了两个关键失败模式：
    1.  **时间插值伪影（Ghosting）**：当渲染两个输入帧之间的时间点时，模型会生成来自两个相邻输入视图的、空间位置不一致的重复高斯，产生鬼影效应。
    2.  **视角依赖偏差（View-dependent Bias）**：模型过度依赖时间上最接近的输入视图来生成高斯，无法有效利用远离当前时间点的全局信息，导致遮挡区域重建质量极差。
- **根本原因**：论文指出，这些问题源于所有现有前馈4D方法的共同设计缺陷——**逐像素高斯预测**。这种设计赋予了模型过大的表示能力，使其容易过拟合于输入视图，而无需真正学习场景的全局运动。在平滑的动态视频中，仅仅插值相邻帧的像素高斯就能获得不错的渲染分数，模型缺乏学习底层真实运动或几何结构的驱动力。
- **研究意义**：解决该问题对于推动AR/VR、机器人、内容创作等需要可扩展、无需逐场景优化的动态场景理解的应用至关重要。提出一种能够学习**全局运动**且**高度紧凑**的4D表示是该文的核心目标。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：摒弃逐像素预测，采用**可学习的查询令牌（Query Tokens）** 架构。通过一组紧凑的、以目标时间戳为条件的查询令牌，聚合整个输入视频序列的时空信息，从而解码出代表全局动态场景的3D高斯。这种架构上的瓶颈迫使模型学习全局运动，而非拟合单个帧。

- **关键技术细节**：
    1.  **紧凑的4D高斯表示**：
        - 使用VGGT等几何基础模型提取多帧特征。
        - 引入**N个（例如2048个）可学习的查询令牌（Q）**。
        - **时间条件嵌入**：将目标时间戳转换为正弦位置编码，并注入到查询令牌和输入帧特征中。这使得模型能够区分不同帧的时间信息，并针对特定目标时间解码高斯。
        - 查询令牌与特征通过一个带有完全自注意力的**Transformer解码器**进行交互。这种设计迫使每个令牌聚合全局时空信息，而不是某个特定像素的信息。
        - 最后，将精炼后的查询令牌通过一个简单的MLP解码为3D高斯（包括位置、不透明度、协方差、颜色）。

    2.  **渲染增强模块**：
        - 为了解决因Gaussian数量少而可能损失的细节，引入了一个基于**视频扩散模型（VDM）** 的渲染增强模块。

    3.  **特征提升（Feature Lifting）**：
        - 利用C4G解码器中查询令牌自学习到的**注意力模式**，可以复用其查询和键（keys），仅训练少量的值投影层，将任意视觉基础模型（如DINOv3、LSeg）的2D特征提升到3D高斯上。
        - 这就诞生了一个**4D特征场**，每个高斯不仅携带几何和外观信息，还携带了语义或追踪等高级特征，从而支持点跟踪和动态场景理解等下游任务。

    4.  **损失函数**：
        - **光度损失** (MSE + LPIPS)：保证渲染质量。

    输入一个单目视频序列 -> 使用视觉编码器提取每帧特征，并注入时间戳嵌入 -> 将一组可学习的查询令牌也注入目标时间戳嵌入 -> 将所有标记（特征+查询）送入Transformer解码器 -> 通过自注意力机制，查询令牌从所有输入帧的特征中聚合信息 -> 解码器输出精炼后的查询令牌 -> 通过简单的MLP头解码为N个3D高斯（位置由目标时间戳调制） -> 用可微光栅化渲染出新视角图像 -> 用扩散模型进一步增强图像质量。通过迭代不同目标时间戳，即可获得完整的4D动态场景。

### 8. 不足与局限

- **应用限制**：
    - 模型只能重建在输入帧中**至少被观察过一次**的区域。对于极端视角或完全遮挡的区域，会产生**空洞**。虽然扩散模型可以填充，但填充内容可能是幻觉。
    - 模型仅训练于**时间插值**任务，因此无法准确预测输入时间范围**之外**的（未来）场景。
- **实验覆盖偏差风险**：
    - 训练和评估主要基于合成数据集（Spring, Kubric）和一些受控的室内/室外场景。在**真实世界、高度非刚性、拓扑剧烈变化**的复杂动态场景（如揉纸、水流、火焰）上的泛化能力有待进一步验证。
    - 与其它前馈方法一样，依赖于预训练视觉特征提取器（VGGT）的质量，该模型的性能上限在一定程度上受限于此。
- **其他潜在不足**：
    - 虽然高斯数量极少，但解码过程（Transformer + MLP）的计算开销可能不适用于需要极高帧率（如1000+ FPS）的实时应用。
    - VDM增强模块虽然提升了视觉质量，但它本身是一个计算量较大的扩散模型，处理速度会相对较慢。
